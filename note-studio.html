Note Studio

A browser-based Anki note type editor. Import .colpkg, .apkg, or .anki2 files. Everything runs locally — no data leaves your device.

• Fields — rename, reorder, add, remove fields
• Templates — edit front/back HTML + CSS
• Preview — live card rendering with field inputs
• Notes — browse notes, click to load into preview
• Media — view and download embedded media files
• Export — download as JSON, CSV, or .apkg
Built from the Anki open-source codebase (AGPL-3.0)
